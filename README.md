# Yo idon

Yeoman generator for starting sketch HTML, CSS, and JavaScript

## Features

- CSS Autoprefixing
- Automagically complie Jade, Sass, CoffeeScript
- Built-in server with Browser-sync

## Getting Started

    git clone git@github.com:shgtkshruch/generator-idon.git
    cd generator-idon
    npm link
    
## Usage

    mkdir myproject
    cd myproject
    yo idon
